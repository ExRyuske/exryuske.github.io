DualSense in RDR2

1. Install DSX
2. Download Script Hook RDR2
ㅤCopy ".dll" files to RDR2 dir
3. Download Script Hook RDR2 .NET and its requirements
ㅤCopy files to RDR2 dir
4. Download DSX4RDR2
ㅤCreate "Scripts" in RDR2 dir
ㅤCopy ".dll" files to created folder
5. Download Lenny's Mod Loader
ㅤCopy files from "ModLoader" folder into RDR2 dir
6. Download PlayStation Icons Replacement
ㅤExtract folder to "lml" folder in RDR2 dir
7. Start "DSX" and set emulation to "Xbox360"

Games on Another Drive

1. Create a directory and grant access to it
ㅤsudo mkdir /games
ㅤsudo chmod -R 777 /games
2. Add to /etc/fstab
ㅤUUID="disk uuid" /games ext4 defaults 0 0
3. Restart PC
