Windows 11

1. Settings > System > Display > Graphics > Default graphics settings > Enable Optimization for windowed games
2. Settings > Bluetooth & devices > Mouse > Additional mouse settings > Pointer Options > Disable Enchance pointer precision

DualSense in RDR2

1. Install DSX
2. Download archive
3. Unzip 002.zip to RDR2 directory
3. Start DSX, set emulation to "Xbox360"

Games on another drive

1. Create a directory sudo mkdir /games
2. Grant access to directory sudo chmod -R 777 /games
3. Add to /etc/fstab UUID="foo" /games ext4 defaults 0 0
4. Reboot PC

Lock cursor in Xbox games

1. Press Win + R
2. Enter regedit
3. Go Computer\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\System
4. Change 0 to 1 in the EnableLUA

Tissot install custom ROM

1. Download ROM and this files
2. Enable OEM Unlock and USB Debugging
3. Unzip boot-recovery.img to the ADB RUN bin folder
4. Open ADB RUB and enter "A"
5. Enter adb reboot bootloader
6. Enter fastboot oem unlock
7. After rebooting, repeat steps 2 and 5
8. Enter fastboot flash boot boot-recovery.img
9. Enter fastboot reboot
10. Wipe dalvik, system, data
11. Copy ROM and Magisk to device
12. Install ROM in the active slot and inject TWRP
13. Reboot to recovery and install Magisk
14. Reboot into system

VAC was unable to verify ur game session

1. Check the system date and time
2. Close the steam
3. Go to the steam folder C:\Program Files (x86)\Steam
4. Delete Steam.dll and SteamUI.dll
5. Press Win + R
6. Enter C:\Program Files (x86)\Steam\bin\SteamService.exe /repair
7. Open Steam
8. Open the steam download settings and clear the cache
9. Check the integrity of the Counter-Strike files

Xbox app stuck on syncing data

1. Open Task Manager
2. Find Gaming Services
3. End this task
4. Restart the game
