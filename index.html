Tissot install custom ROM
More...

1. Download ROM & this files
2. Enable OEM Unlock and USB Debugging
3. Unzip boot-recovery.img to the ADB RUN bin folder
4. Open ADB RUB and enter A
5. Enter adb reboot bootloader
6. Enter fastboot oem unlock
7. After rebooting, repeat 2, 5
8. Enter fastboot flash boot boot-recovery.img
9. Enter fastboot reboot
10. Wipe dalvik, system, data
11. Copy ROM and Magisk to device
12. Install ROM in the active slot and inject TWRP
13. Reboot to recovery and install Magisk
14. Reboot into system
